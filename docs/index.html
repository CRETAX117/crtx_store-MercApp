CRTX Store - MercApp

Catalogo de productos con carrito de compras

Brandon Cardenas — Aplicaciones Web — N1 Cuenca

Enlaces del proyecto

  • Frontend en Netlify (pendiente)
  • API en Railway (pendiente)
  • Repositorio en GitHub

Arquitectura

Netlify (Vue 3)  →  Railway (Flask API)  →  MongoDB Atlas (M0)
 *.netlify.app        *.railway.app           cluster0.mongodb.net

Endpoints

MetodoRutaQue hace
GET/healthEstado del servidor
GET/api/productsLista productos (?search= y ?category=)
GET/api/products/:idUn producto por ID
POST/api/productsCrear producto
PUT/api/products/:idEditar producto
DELETE/api/products/:idEliminar producto
GET/api/categoriesLista categorias

Tecnologias

  • Vue 3 + TypeScript + Vite
  • Flask + PyMongo
  • MongoDB Atlas (cluster M0)

Como correr en local

Backend:

cd backend
pip install -r requirements.txt
python seeds/seed.py
python app.py

Frontend:

cd frontend
npm install
npm run dev

## Descripción

Aplicación web tipo catálogo de productos. Va a tener un backend con Flask que expone una API REST y un frontend con Vue 3 que consume esa API.

La idea es poder ver productos, buscarlos, filtrar por categoría, y manejar un carrito de compras.
